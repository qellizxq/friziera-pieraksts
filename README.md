# Friziera pieraksts

Šī ir interaktīva mājaslapa friziera pakalpojumu rezervācijai.

## Funkcionalitāte
- Lietotājs izvēlas pakalpojumu.
- JavaScript nolasa ievadi.
- Tiek pārbaudīts, vai pakalpojums ir izvēlēts.
- Cena tiek iegūta no objekta.
- Rezultāts tiek dinamiski attēlots lapā bez pārlādes.

## Lietošanas instrukcija
1. Izvēlies pakalpojumu no saraksta.
2. Nospied pogu "Rezervēt".
3. Parādīsies cena vai kļūdas paziņojums.
